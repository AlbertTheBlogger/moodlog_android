# 📝 微光日记 - Android 应用开发期末作业

> **微光日记** 是一款专注于记录每日正能量时刻的应用，旨在帮助用户捕捉生活中的小确幸。本项目作为《移动应用开发》课程的期末作业，实现了从设计到实现的完整流程。

---

## 🎯 项目来源与目标

### 来源
- **课程要求**：本项目是《移动应用开发》课程的期末作业，要求学生独立完成一个具有实际意义的 Android 应用。
- **现实痛点**：现代快节奏生活中，人们往往忽略了身边的美好瞬间。本应用鼓励用户每天记录下这些美好的时刻，提升生活的幸福感。

### 目标
- 提供简单易用的界面，方便用户快速记录心情和事件。
- 通过日历视图直观展示用户的记录习惯。
- 随机展示过往的日记条目，帮助用户回顾过去的美好时光。

---

## ⚙️ 技术特点

| 类别         | 技术栈 / 实现方式                                                                 |
|--------------|----------------------------------------------------------------------------------|
| **开发语言**   | Java（经典、稳定）                                                                |
| **UI 框架**    | XML + Activity + Fragment（传统布局系统）                                         |
| **数据存储**   | JSON 文件（内部存储），Gson 进行序列化/反序列化                                    |
| **日期处理**   | `java.util.Date` 和自定义工具类进行日期格式化                                      |
| **动画效果**   | 系统默认涟漪效果，淡入淡出等基础反馈                                               |
| **主题系统**   | Material 3 动态配色 + 自定义颜色集                                                 |

### 核心功能亮点
- ✨ **编辑日记**：支持多行文本输入，选择情绪标签，保存日记。
- 🌙 **日历查看**：按月份浏览日记记录情况，不同颜色代表不同的日记数量。
- 💾 **本地持久化**：所有数据保存在设备本地，保障隐私与离线可用性。
- 🎨 **定制化 UI**：采用橙色主色调（`#FF9800`），界面清爽直观。
